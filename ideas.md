# Brainstorm: Jogo Psicodélico Interativo - Abordagens de Design

## Contexto
Um jogo web que começa simples e confortável, evoluindo progressivamente para algo estranho, invasivo e psicológico. A experiência deve desconfortar e fazer o usuário refletir sobre suas próprias escolhas.

---

## Resposta 1: Brutalism Digital + Glitch Progressivo
**Probabilidade: 0.08**

### Design Movement
Brutalism Digital com influências de Datamoshing e Arte Generativa

### Core Principles
1. **Autenticidade Crua**: Sem polimento, mostrando o "código" e a "máquina" por trás
2. **Progressão Visual Destrutiva**: Começar limpo, terminar corrompido
3. **Tipografia como Ferramenta de Estranheza**: Mudanças de fonte e tamanho refletem instabilidade mental
4. **Espaço Negativo Agressivo**: Muito vazio, pouco conteúdo, criando sensação de isolamento

### Color Philosophy
- **Fase 1-2**: Monocromático (branco/cinza sobre preto) — clareza industrial
- **Fase 3-4**: Injeção de verde/ciano (referência a terminal/hacking) — invasão
- **Fase 5**: Vermelho sangue + distorção cromática — caos total
- **Psicologia**: Começar com a "pureza" da máquina, terminar com sua "corrupção"

### Layout Paradigm
- Começar com grid rígido 1x1 (um elemento por tela)
- Fase 2: Elementos começam a se desalinhar levemente
- Fase 3: Múltiplos elementos competem por espaço
- Fase 4-5: Sobreposição caótica, texto saindo da tela

### Signature Elements
1. **Cursor Piscante**: Referência a terminal antigo, piscando mais rápido conforme avança
2. **Linhas de Scan**: Efeito de CRT/monitor antigo, intensificando-se
3. **Glitch Boxes**: Retângulos distorcidos que "corrompem" o conteúdo

### Interaction Philosophy
- Cliques produzem "lag" visual (delay antes de resposta)
- Botões mudam de cor/posição sutilmente a cada interação
- Hover effects criam "artefatos" ao redor dos elementos
- Feedback é sempre ligeiramente atrasado ou inesperado

### Animation
- Transições lentas no início (200-300ms)
- Velocidade aumenta conforme fases avançam
- Tremores sutis de tela (transform: translateX/Y com valores aleatórios)
- Fade-in/fade-out com efeito de "dissolução" (opacity + blur simultâneos)

### Typography System
- **Display**: IBM Plex Mono (monospace, rígido, industrial)
- **Body**: Space Mono (monospace, mantém a frieza)
- Tamanhos: Começar em 16px, aumentar para 24px+ nas fases finais
- Peso: Alternar entre regular e bold para criar ritmo visual

## Resposta 2: Minimalismo Perturbador + Efeitos Sutis
**Probabilidade: 0.07**

### Design Movement
Minimalismo Contemporâneo com Psicologia Gestalt Invertida

### Core Principles
1. **Menos é Mais, Mas Mais é Perturbador**: Começar com absoluto minimalismo, adicionar elementos sutis que desconfortam
2. **Espaço como Personagem**: O vazio não é vazio, é presença
3. **Tipografia Elegante Corrompida**: Fontes belas que gradualmente se deformam
4. **Paleta Limitada, Impacto Máximo**: Poucas cores, cada mudança é significativa

### Color Philosophy
- **Fase 1-2**: Branco puro (rgb 255,255,255) sobre fundo preto — pureza
- **Fase 3**: Cinza introduzido (rgb 128,128,128) — dúvida
- **Fase 4**: Bege/sepia (rgb 200,180,160) — nostalgia perturbadora
- **Fase 5**: Vermelho muito pálido (rgb 255,200,200) — sangue diluído
- **Psicologia**: Degradação gradual da pureza para a corrupção

### Layout Paradigm
- Começar com texto centralizado, linha única
- Fase 2: Texto se divide em 2 linhas
- Fase 3: Múltiplas colunas, mas ainda organizado
- Fase 4: Assimetria crescente
- Fase 5: Caos controlado (elementos fora do grid)

### Signature Elements
1. **Linha Horizontal Divisória**: Separa "acima" (consciente) de "abaixo" (subconsciente)
2. **Espaço em Branco Respirante**: Padding que aumenta/diminui sutilmente
3. **Ponto de Foco Invisível**: Um elemento que não está lá, mas você sente sua ausência

### Interaction Philosophy
- Cliques produzem expansão/contração de espaço branco
- Hover revela texto escondido (opacity 0 → 1)
- Botões desaparecem e reaparecem
- Feedback é visual e silencioso (sem som)

### Animation
- Transições muito lentas (400-600ms) — desconfortavelmente lenta
- Easing: ease-in-out para criar sensação de "peso"
- Opacity changes como principal ferramenta de animação
- Scale transformations sutis (0.98 → 1.02)

### Typography System
- **Display**: Playfair Display (elegante, clássico, perturbador)
- **Body**: Lora (serif, leitura confortável que fica estranha)
- Tamanhos: Começar em 18px, reduzir para 12px nas fases finais (inversão psicológica)
- Peso: Manter light/regular, nunca bold (falta de força)

## Resposta 3: Psicodelia Literal + Distorção Progressiva
**Probabilidade: 0.09**

### Design Movement
Psicodelia Digital com Influências de Vaporwave e Glitch Art

### Core Principles
1. **Cores Vibrantes Que Machucam**: Começar vibrante, terminar em tons que causam desconforto visual
2. **Distorção é Narrativa**: Cada glitch conta parte da história
3. **Simetria Quebrada**: Começar simétrico, terminar assimétrico
4. **Movimento Constante**: Nada está parado, tudo respira/pulsa

### Color Philosophy
- **Fase 1**: Cores pastel suaves (rosa, azul claro, amarelo) — conforto
- **Fase 2**: Cores mais saturadas (magenta, ciano, amarelo neon) — estranheza
- **Fase 3**: Cores complementares em conflito (roxo + amarelo) — tensão
- **Fase 4**: Cores dessaturadas + vermelho (cinza + sangue) — morte
- **Fase 5**: Preto com neon (branco + ciano) — vazio com vida
- **Psicologia**: Progressão de harmonia para dissonância cromática

### Layout Paradigm
- Começar com simetria perfeita (centro)
- Fase 2: Rotações sutis (1-3 graus)
- Fase 3: Elementos em perspectiva 3D
- Fase 4: Sobreposição com opacity variável
- Fase 5: Explosão radial (elementos saindo do centro)

### Signature Elements
1. **Gradientes Animados**: Cores se movem pela tela como lava
2. **Fractais Sutis**: Padrões que se repetem em escalas diferentes
3. **Distorção de Onda**: Efeito de "água" que deforma o conteúdo

### Interaction Philosophy
- Cliques produzem explosões de partículas coloridas
- Hover cria halos/auras ao redor dos elementos
- Botões "respiram" (scale up/down continuamente)
- Feedback é visual, colorido e imediato

### Animation
- Transições rápidas (100-200ms) — energia
- Easing: cubic-bezier para movimentos "fluidos" e "orgânicos"
- Rotações contínuas (rotate 360deg em 20s)
- Blur/filter effects que mudam constantemente
- Keyframe animations para padrões pulsantes

### Typography System
- **Display**: Fredoka One (redonda, playful, perturbadora em contexto sombrio)
- **Body**: Quicksand (suave, mas estranha em tamanhos grandes)
- Tamanhos: Começar em 20px, aumentar para 32px+ (amplificação)
- Peso: Alternar light/bold para criar ritmo visual agressivo
- Transformações: Skew, rotate, scale aplicadas ao texto

## Decisão Final
**Escolhido: Resposta 1 - Brutalism Digital + Glitch Progressivo**

### Justificativa
Esta abordagem oferece:
- **Autenticidade**: Sensação de que o jogo é "real" e "máquina"
- **Progressão Clara**: Cada fase tem mudança visual óbvia
- **Impacto Psicológico Forte**: A degradação visual reflete degradação mental
- **Facilidade Técnica**: Glitches são mais fáceis de implementar com CSS/JS
- **Coerência Narrativa**: Começa como terminal (controle), termina como caos (perda de controle)

### Paleta Final
```
Fase 1-2: #FFFFFF (texto) sobre #000000 (fundo)
Fase 3: #00FF00 (verde terminal) + #00FFFF (ciano)
Fase 4: #FF0000 (vermelho) + distorção cromática
Fase 5: #000000 (preto) + #FFFFFF (branco neon)
```

### Tipografia
- **Monospace**: IBM Plex Mono para todo o texto
- **Cursor**: Piscante, referência a terminal antigo
- **Tamanho**: 16px → 24px+ conforme avança

### Elementos Assinatura
1. Cursor piscante
2. Linhas de scan (CRT effect)
3. Glitch boxes
4. Tremores de tela
5. Efeito de "lag" visual
